# **CINEMA - TICKETS - JAVA**
This repository contains a cinema ticket exercise code built using Java.

## Getting Started

### Prerequisites

- Java JDK 21 or higher
- Any IDE (IntelliJ, Eclipse, VS Code)

### Installation

```bash
git clone https://github.com/akhilkrishnanns/cinema-tickets.git
cd cinema-tickets-java
```
## Features
1. **Purchase Ticket**
2. **i18n implementation**

   
### Sample Workflow
Start running by calling unit test runs.
